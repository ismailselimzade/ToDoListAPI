# To-Do List API

A REST API for managing personal to-do items with user authentication. İstifadəçi autentifikasiyası ilə şəxsi tapşırıqları idarə etmək üçün REST API.

## What I learned / Öyrəndiklərim

- How to implement JWT-based authentication and protect API routes with middleware
- JWT əsaslı autentifikasiyanı necə tətbiq etmək və API route-larını middleware ilə qorumaq

- How to hash and verify passwords securely using a service layer
- Şifrəni servis təbəqəsi vasitəsilə necə təhlükəsiz hash etmək və yoxlamaq

- How to read the logged-in user's identity from JWT claims
- Giriş etmiş istifadəçinin kimliyini JWT claim-lərindən necə oxumaq

- How to filter database records by status and user ownership
- Verilənlər bazası qeydlərini status və istifadəçiyə məxsusluğa görə necə filtirləmək

- How to use DTOs to separate API contracts from database models
- DTO-lardan istifadə edərək API müqavilələrini verilənlər bazası modelindən necə ayırmaq

## Tech stack / Texnologiyalar

C#, ASP.NET Core, Entity Framework Core, SQL Server, JWT
